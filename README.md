# Unix


1. Run 

    ```bash
        $ git clone https://github.com/tttfrfr2/pVimrc.git ~/.vim
        $ sh ~/.vim/bin/vim-plug-installer.sh
        $ cp vimrc ~/.vim/vimrc
    ```

2. Run vim

3. :PlugInstall in your vim
